# Capturas de la app

Coloca aquí las capturas reales de **Yupay Turismo**. Mientras un archivo no exista,
la web muestra automáticamente un *placeholder* de marca (no se rompe nada ni hace
falta tocar código). En cuanto dejes el archivo con el nombre indicado, aparece la
captura real.

| Archivo        | Pantalla                                   | Uso en la web                 |
|----------------|--------------------------------------------|-------------------------------|
| `home.png`     | Pantalla principal / Home                  | Hero (mockup grande)          |
| `visitas.png`  | Registro o historial de visitas            | Sección "Registro de visitas" |
| `dashboard.png`| Gráficos de insights                       | Sección "Dashboard"           |
| `mapa.png`     | Mapa de procedencias                       | Sección "Mapa"                |
| `audio.png`    | Reproductor de audioguías *(opcional)*     | Sección "Audioguías"          |
| `sync.png`     | Sincronización / QR entre dispositivos     | Sección "Sincronización"      |

## Recomendaciones
- **Formato:** PNG o JPG.
- **Orientación:** vertical, ratio ≈ 9:19.5 (p. ej. 1080×2340). Se recortan con
  `object-fit: cover` dentro del marco del teléfono.
- **Nombre exacto:** respeta el nombre de la tabla (en minúsculas).
- Puedes capturar en modo claro u oscuro; con una sola captura por pantalla basta.
